# My Fitness Plan

Personal fitness and nutrition tracking app.

## Deploy to Vercel (free)

1. Upload this entire folder to a GitHub repository
2. Go to vercel.com → Add New Project → Import your repo
3. Leave all settings as default → click Deploy
4. Your app will be live at yourname.vercel.app

## Run locally

```
npm install
npm run dev
```

## Features
- Daily calorie, macro, water & step tracking
- Streak system with flame emojis
- Weight logging with progress chart
- Flexible weekly workout schedule
- Exercise checklists
- Shopping list with auto-categories
- Day history — finish a day and save it permanently
- All data saved in localStorage — persists between visits
